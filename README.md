## Project 1.  Bellabeat Case Study Report
Bellabeat designs smart wellness products for women, including wearables and a mobile app that tracks activity, sleep, and stress. To support data driven product development, I analyzed Fitbit data from 30 users to understand daily behaviour patterns and identify opportunities for Bellabeat to enhance user engagement.
This analysis focuses on activity levels, sedentary behaviour, sleep patterns, and calorie expenditure.
